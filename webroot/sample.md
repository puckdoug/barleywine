### Code Blocks

Here's some Rust code:

```rust
use rocket::{get, launch, routes};

#[get("/")]
fn hello() -> &'static str {
    "Hello, Barleywine!"
}

#[launch]
fn rocket() -> _ {
    rocket::build().mount("/", routes![hello])
}
```

And some JavaScript:

